Habilidad Tejedor de Patrones para OpenClaw.

## Qué Soluciona
Descubre y conecta patrones ocultos en bases de código y documentación.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/pattern-weaver

## Ejemplo
```
Analizar código para detectar problemas de análisis.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
